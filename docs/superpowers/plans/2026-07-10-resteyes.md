# RestEyes 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**状态:大纲已定稿,任务详情分阶段增量补全中(防止单次超长写入)。**

**Goal:** 构建 RestEyes——macOS 26+ (arm64) 菜单栏应用,定时全屏遮罩强制休息,屏蔽所有操作,右下角按配置出现解锁按钮,文本文件配置,GitHub Actions 测试与编译。

**Architecture:** 纯 SPM 工程,Swift + AppKit,零第三方依赖。逻辑层(配置解析、计时状态机)为纯 Swift 可单元测试;UI 层(遮罩、状态栏)只消费状态机回调、只发送动作。所有测试与编译在 GitHub Actions macOS runner 上执行,本地(Linux)只写代码。

**Tech Stack:** Swift 6 / AppKit / Swift Package Manager / XCTest / GitHub Actions (`macos-26` runner)

## Global Constraints

- 目标平台:macOS 26.0+,仅 arm64;`LSMinimumSystemVersion = 26.0`
- 零第三方依赖;无 Xcode 工程文件,纯 `Package.swift`
- `LSUIElement = true`(无 Dock 图标);`CFBundleIdentifier = com.resteyes.app`
- 不需要任何 TCC 权限(辅助功能/输入监控)
- **所有「测试通过/编译成功」结论必须以 CI 运行结果为证据**;本地不运行 swift 工具链
- 每个任务结束:`git push` → `GH_CONFIG_DIR=.gh-config gh run watch <run-id>` 确认绿色后才算完成
- gh 命令一律带 `GH_CONFIG_DIR=.gh-config` 前缀(凭据在仓库内 `.gh-config/`,已 gitignore)
- 配置文件路径:`~/.config/resteyes/config.txt`;解析容错,非法值回退默认,绝不崩溃
- ESC×10(间隔 ≤1.5s)紧急后门任何配置下有效,不可关闭
- Commit message 均以 `Co-Authored-By: Claude Fable 5 <noreply@anthropic.com>` 结尾

## 关键接口契约(各任务实现时以此为准)

```swift
// Config.swift
enum UnlockAfter: Equatable { case seconds(Int); case never }
struct Config: Equatable {
    var workMinutes: Double = 20        // (0, 1440]
    var restMinutes: Double = 3         // (0, 1440]
    var warnSeconds: Int = 10           // [0, 600]
    var unlockAfter: UnlockAfter = .seconds(60)  // [0, 86400] | never
    var message: String = "休息一下,眺望远方 🌿"
    var showCountdown: Bool = true
    static func parse(_ text: String) -> Config          // 纯函数,容错
    static let defaultFileContent: String                 // 带中文注释的默认配置文本
    static func load(from url: URL) -> Config             // 不存在则创建默认文件后解析
    static var defaultURL: URL                            // ~/.config/resteyes/config.txt
}

// BreakScheduler.swift(不 import AppKit;外部每秒调 tick(now:))
enum Phase: Equatable { case working, warning, resting, paused }
final class BreakScheduler {
    init(config: Config, now: Date)
    var onPhaseChange: ((Phase) -> Void)?                 // 相位切换时回调
    var onTick: ((Phase, TimeInterval) -> Void)?          // 每次 tick 回调(当前相位, 剩余秒)
    private(set) var phase: Phase
    private(set) var skipNextArmed: Bool
    var unlockButtonDelay: UnlockAfter                    // 当前配置的解锁按钮时机(透传)
    func tick(now: Date)                                  // 驱动一切状态迁移
    func breakNow(now: Date)                              // 立即休息(跳过预警)
    func toggleSkipNext()                                 // 跳过下次休息(一次性,可取消)
    func pause(now: Date)                                 // 暂停 1 小时
    func resume(now: Date)                                // 提前恢复;工作计时重开
    func unlock(now: Date)                                // 解除休息(按钮/ESC 后门共用)
    func reload(config: Config, now: Date)                // 重载配置;工作计时按新值重开
    func systemDidWake(sleptFor: TimeInterval, now: Date) // 睡眠唤醒分支处理
}

// OverlayController.swift
final class OverlayController {
    var onUnlockRequested: (() -> Void)?                  // 解锁按钮 或 ESC×10 触发
    func showWarning(secondsRemaining: Int)               // 预警浮窗(主屏右下角)
    func hideWarning()
    func showRest(config: Config)                         // 全屏遮罩 + kiosk 开启
    func updateRest(remaining: TimeInterval, unlockVisible: Bool)
    func hideRest()                                       // 遮罩移除 + kiosk 恢复
}

// StatusItem.swift
final class StatusItemController {
    var onBreakNow, onToggleSkipNext, onPauseToggle,
        onOpenConfig, onReloadConfig, onQuit: (() -> Void)?
    func update(phase: Phase, remaining: TimeInterval, skipNextArmed: Bool, paused: Bool)
}
```

## 任务大纲(三个阶段)

### 阶段 A:地基(纯逻辑 + CI 闭环)

- **Task 1:SPM 工程骨架 + CI workflow 跑绿**
  `Package.swift`、最小 `main.swift`(可编译占位)、空测试、`.github/workflows/build.yml`(swift test → swift build)。推送并确认首个 CI 全绿——之后每个任务都有测试闭环。

- **Task 2:Config 配置解析(TDD)**
  `ConfigTests.swift` 全用例(默认值/注释空行/未知键/非法值回退/never/小数/边界)→ `Config.swift` 实现 → CI 绿。

- **Task 3:BreakScheduler 计时状态机(TDD)**
  `BreakSchedulerTests.swift` 全用例(完整周期/warn=0/skipNext 一次性/pause-resume/breakNow/unlock/reload/睡眠唤醒三分支)→ `BreakScheduler.swift` 实现 → CI 绿。

### 阶段 B:UI 层(AppKit,编译验证 + 手动验收)

- **Task 4:OverlayController 全屏遮罩**
  每屏 shielding 窗口、kiosk presentationOptions、message+倒计时、右下角解锁按钮(unlock_after 三种模式)、ESC×10 后门、预警浮窗、屏幕增减监听、每秒重新激活。CI 编译绿(无单测,行为进手动验收清单)。

- **Task 5:StatusItemController 状态栏菜单**
  眼睛模板图标、七项菜单(倒计时显示/立即休息/跳过下次/暂停⇄恢复/打开配置/重载配置/退出)。CI 编译绿。

- **Task 6:main.swift 组装 + 睡眠唤醒**
  AppDelegate 接线五个模块;每秒 Timer 驱动 `tick`;NSWorkspace 睡眠/唤醒/锁屏通知 → `systemDidWake`;每工作周期开始自动重读配置;单实例保护。CI 绿。

### 阶段 C:交付

- **Task 7:CI 打包 RestEyes.app + artifact**
  workflow 增加:组装 .app(`Resources/Info.plist`)、ad-hoc codesign、zip、upload-artifact。下载 artifact 验证结构。

- **Task 8:Release + README**
  tag `v*` 触发 `gh release create` 附 zip;README(安装含右键打开与 xattr、配置说明、手动验收清单)。打 `v0.1.0` 验证 Release 产生。
