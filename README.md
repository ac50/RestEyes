# RestEyes

macOS 菜单栏应用:定时全屏遮罩,强制休息眼睛、起身活动。

- 定时黑屏(或显示自定义文字),覆盖所有窗口、所有 Space、所有显示器
- 休息期间屏蔽键盘与应用切换(Cmd+Tab / Cmd+Q / Cmd+Opt+Esc 等均无效)
- 右下角「解锁」按钮,出现时机可配置(立即 / 延迟 N 秒 / 永不)
- 紧急后门:连按 ESC 10 次强制解锁,任何配置下有效
- 黑屏前 10 秒角落倒计时预警(可配置/可关)
- 休息结束后自动锁屏(同 Ctrl+Cmd+Q 效果,可配置关闭;手动解锁不触发)
- 休息中主动锁屏/合盖,解锁后直接回到工作(可配置关闭)
- 开机自启动(默认开启,可配置关闭)
- 无 GUI 设置界面:文本文件配置 + 状态栏菜单
- 仅支持 Apple Silicon,macOS 26+

## 安装

1. 从 [Releases](https://github.com/ac50/RestEyes/releases) 下载 `RestEyes.zip`,解压
2. 把 `RestEyes.app` 拖入「应用程序」
3. 首次运行:**右键 → 打开**(应用未做付费签名公证)。若仍被拦截,执行:
   `xattr -cr /Applications/RestEyes.app`
4. 状态栏出现眼睛图标即在运行。首次运行会自动注册开机自启(默认开启,配置 `launch_at_login = off` 可关闭;注册时系统会通知"RestEyes 已添加为登录项")

## 配置

配置文件:`~/.config/resteyes/config.txt`(首次启动自动生成)。
改完后在状态栏菜单点「重新加载配置」立即生效;每个工作周期开始时也会自动重读。

| 键 | 默认 | 说明 |
|---|---|---|
| `work_minutes` | `20` | 工作时长(分钟,可用小数) |
| `rest_minutes` | `3` | 休息时长(分钟,可用小数) |
| `warn_seconds` | `10` | 黑屏前预警秒数,`0` = 不预警直接黑屏 |
| `unlock_after` | `60` | 解锁按钮出现时机:秒数;`0` = 一开始就显示;`never` = 永不显示 |
| `message` | 休息一下,眺望远方 🌿 | 遮罩文字,留空 = 纯黑屏(不能含 `#`) |
| `show_countdown` | `on` | 遮罩上显示剩余时间(`on`/`off`) |
| `lock_after_rest` | `on` | 休息自然结束后进入系统锁屏;手动解锁/唤醒解除不触发(`on`/`off`) |
| `wake_ends_rest` | `on` | 睡眠/锁屏后唤醒解锁时直接结束休息回到工作(`on`/`off`) |
| `launch_at_login` | `on` | 开机自动启动;在「系统设置 → 登录项」手动关闭后以系统设置为准(`on`/`off`) |

`#` 起为注释;未知键忽略;非法值自动回退默认值,不会崩溃或锁死。

## 状态栏菜单

距下次休息倒计时 / 立即休息 / 跳过下次休息 / 暂停 1 小时 ⇄ 恢复 / 打开配置文件 / 重新加载配置 / 退出

## 已知边界

电源键、Ctrl+Cmd+Q 系统锁屏、强制重启无法被用户态应用屏蔽。睡眠或锁屏超过一次休息时长,唤醒后视为已休息,工作计时重新开始。
纯"熄屏"若未触发系统锁屏(未开启"立即要求密码")则检测不到;开启 wake_ends_rest 时,休息刚开始就锁屏再解锁可跳过休息,由用户自律。
在「系统设置 → 登录项」手动关闭开机自启后,应用不会强行重新打开;想恢复可在系统设置里开启,或把配置切为 off 再改回 on 后重载。

## 手动验收清单

- [ ] 到点后预警浮窗出现,倒计时结束全屏黑屏
- [ ] 遮罩覆盖所有显示器、所有 Space、全屏应用之上
- [ ] 遮罩期间 Cmd+Tab / Cmd+Q / Cmd+H / Cmd+Opt+Esc 无效,Dock 与菜单栏隐藏
- [ ] `unlock_after` 三种模式:`0` 立即出现、`60` 一分钟后出现、`never` 永不出现
- [ ] ESC 连按 10 次强制解锁
- [ ] 休息时间到自动解锁回到工作计时
- [ ] 休息中插拔显示器,新屏幕立即被遮罩
- [ ] 菜单七项各自工作;跳过下次生效一次后自动清除
- [ ] 睡眠/锁屏超过休息时长后唤醒,工作计时重新开始
- [ ] `message` 留空为纯黑屏;`show_countdown = off` 无倒计时
- [ ] 深浅色菜单栏下图标均清晰
- [ ] 休息自然结束后进入系统锁屏;点「解锁」或 ESC×10 结束则不锁;`lock_after_rest = off` 时不锁
- [ ] 休息中锁屏再解锁:默认直接回到工作且不触发锁屏;`wake_ends_rest = off` 时遮罩继续倒计时
- [ ] 首次运行后「系统设置 → 通用 → 登录项」出现 RestEyes;`launch_at_login = off` 并重载配置后消失

## 开发

纯 SPM 工程(Swift + AppKit,零依赖)。测试与编译均在 GitHub Actions 的 macOS runner 上运行:push 触发 `swift test` + 打包,推 `v*` tag 自动发 Release。
